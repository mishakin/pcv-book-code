This software implements demos and figures shown in the book

Photogrammetric Computer Vision
Statistics, Geometry, Orientation and Reconstruction
Wolfgang Förstner and Bernhard P. Wrobel
Springer 2016

For any question or comment please contact
Wolfgang Förstner 
wfoerstn@uni-bonn.de
http://www.ipb.uni-bonn.de/people/wolfgang-forstner/

-------------------------------------------------------------------

Tested under Matlab 2010b and 2016b. 
Requires the following Matlab toolboxes

    Image Processing Toolbox
    Statistics and Machine Learning Toolbox

-------------------------------------------------------------------

Directories to be installed, of individual parts

02-Probability
03-Testing
04-Estimation
05-08-Geometry
10-Uncertain-Geometry
12-Single-View-Geometry
13-Two-View-Geometry
16-Surface-Reconstruction
General-Functions
