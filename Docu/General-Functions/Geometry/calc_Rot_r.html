calc_Rot_r

PURPOSE

% Cayley rotation matrix

SYNOPSIS

function R = calc_Rot_r(r)

DESCRIPTION 

% Cayley rotation matrix
 R = (I-S_r)^-1*(I+S_r)
 see PCV (8.62)
 
 Usage:
   R = calc_Rot_r(r)
 
   r - double 3x1
   R - double 3x3, 3D rotation matrix (rotates by 2|r|)
