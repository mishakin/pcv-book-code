calc_S

PURPOSE

% skew matrix of 3-vector

SYNOPSIS

function S = calc_S(x)

DESCRIPTION 

% skew matrix of 3-vector
 see PCV (A.34)
 
 Usage:
   S = calc_S(x)

   x - 3x1 vector
   S - 3x3 skew matrix

 let x = [a b c], than
     S = [  0 -c  b |
         | -c  0 -a |
         | -b  a  0 ]

 wfoerstn@uni-bonn.de 

 See also calc_S_reduced, calc_v_from_S
