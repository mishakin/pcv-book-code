sugr_angle_Line_2D

PURPOSE

% Get angle between two 2D lines 

 angle = sugr_angle_Line_2D(l,m)

 * l, m   2D line, struct {l.h,l.Crr}

 * angle.a  = angle between lines, from l to m in [0,2*pi)%
        .sa = standard deviation angle negativ if one line is infinte%
