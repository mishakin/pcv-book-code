sugr_get_Euclidean_Point_2D

PURPOSE

% Get Euclidean coordinates with CovM

SYNOPSIS

function [e,Cee] = sugr_get_Euclidean_Point_2D(x)

DESCRIPTION 

% Get Euclidean coordinates with CovM

 [e,Cee] = sugr_get_Euclidean_Point_2D(x)
 
 if finite point:
    e   = Euclidean coordinates
    Cee = CovM of e
 else zeros
