sugr_get_Euclidean_Point_3D

PURPOSE

% Get Euclidean coordinates of 3D point with CovM

SYNOPSIS

function [e,Cee] = sugr_get_Euclidean_Point_3D(x)

DESCRIPTION 

% Get Euclidean coordinates of 3D point with CovM

 [e,Cee] = sugr_get_Euclidean_Point_3D(x)

 if finite point:
      e   = 3x1 Euclidean coordinates
      Cee = 3x3 CovM of e

 else zeros
